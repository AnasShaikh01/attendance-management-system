# 📌 Attendance Management System

A modern web-based **Attendance Management System** built using **PHP, Tailwind CSS, and MySQL** to efficiently manage and track student attendance.

This system allows teachers and administrators to record, manage, and analyze attendance through a clean and responsive interface.

---

## 🚀 Features

- 👨‍🏫 Admin & Teacher Authentication
- 📅 Mark & Update Student Attendance
- 📊 Automatic Class Average Calculation
- 📈 Attendance Statistics Dashboard
- 🗂 Student Management (Add / Edit / Delete)
- 📚 Subject & Class Management
- 📱 Fully Responsive UI using Tailwind CSS
- 🛡 Secure Login System

---

## 🛠 Tech Stack

- **Frontend:** Tailwind CSS  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server Environment:** XAMPP / Apache

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/AnasShaikh01/attendance-management-system.git
```

### 2️⃣ Move Project to XAMPP

Place the folder inside:

```
C:\xampp\htdocs\
```

### 3️⃣ Start XAMPP

- Start **Apache**
- Start **MySQL**

### 4️⃣ Import Database

- Open **phpMyAdmin**
- Create a new database (e.g., `attendance_system`)
- Import the provided `.sql` file

### 5️⃣ Configure Database Connection

Update database credentials inside:

```
config/database.php
```

### 6️⃣ Run the Project

Open your browser and go to:

```
http://localhost/attendance-management-system/
```
